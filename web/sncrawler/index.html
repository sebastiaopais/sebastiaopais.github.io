Our Crawler

Our crawler was developed in Java programming language and our interface was done with Java Swing. To get started, you only need to visit the social networks developer pages our crawler works with, and follow the steps on the respective pages to get an access token. Then put that access token on our application in the tabs of each social network you want to crawl and press start.
